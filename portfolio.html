About

I'm a User Interface Designer and Developer with a background in Graphic Design, User Experience Design, and Animation. Growing up playing video games, watching 3D animated movies, and surfing the web, I have always been amazed and excited with digital technology. All this experience influenced me to pursue my dream in something related to web development or animation.
My long adventure took me from a creative with some coding experience to someone who welcomes challenges and finding bugs in an application. As a graphic designer I'm obsessed with the look and details, as an animator I learned how to use motion and interaction to design a better application, as a User Experience Designer I learned to count in my users, and as a developer, I found out about endless possibilities. I find great joy in making things that are unique and impactful.

Projects

Chat Station is chatting app where each user can create a chat room and chat with other users. Chat Station is Built with React.js and Google Firebase. Also, users can log in and log out with their Google account.

Demo Github

Pumadoro is a task management app based on The Pomodoro Technique. each user has 25 minutes to complete a task or reset the time over. After each task, the user has an option to take a 5 minutes break. The break time would increase to 30 minutes after every four completed tasks. Pumadoro is powered by React.js and Google Firebase.

Demo Github

Task Devil app is a To-do list that allows users to create and delete tasks. The user can tick-off tasks and visit the completed tasks in the Task History tab. The user has the option to tick-off completed tasks in the Task History to make them active again. Task Devil deletes uncompleted task after seven days. This application is made with React.js and Google firebase.

Demo Github

Artmates provides a live chat between a museumgoer and a tour guide. Unlike the other apps in the market that feed museum's visitors with monotonous facts, Artmates' certified tour guides help people learn art by telling them amazing stories.
